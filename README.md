Manage Appointments & Services for Hairdressers.


## Tech Stack

- Typescript
  - Prettier for code formatting
  - Jest for unit tests
- Supabase
- Next.js

## Prerequisites

- Node JS (v22 LTS)
- Apple ID

## Installation & setup

```bash
# install all the things...
./setup.sh

# start em up...
./start.sh
```
